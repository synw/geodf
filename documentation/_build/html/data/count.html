Count¶

Nulls¶

countNulls

Count the null values in a column.

Positional parameter:

columnName

String: the name of the column to count nulls in

Named parameter:

nullValues

List<dynamic>: list of values considered. Default: [null, "null", "nan", "NULL"]

final int nulls = df.countNulls("speed", nullValues= <dynamic>[null]);

Zeros¶

Count zeros in a column

final int zeros = df.countZeros("speed", zeroValues= <dynamic>[0,"0"]);

The zeroValues parameter is optional. It is a list of values considered zeros. Default values: [0]
